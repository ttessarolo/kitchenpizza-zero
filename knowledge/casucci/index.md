## Come usare questa Knowledge Base

Questa knowledge base contiene il testo completo del libro "La Pizza è un Arte" di Fabrizio Casucci,
organizzato per argomenti. Ogni argomento è un file markdown separato nella cartella `arguments/`.

**Per trovare informazioni specifiche:**
1. Consulta l'indice qui sotto per identificare l'argomento di interesse
2. Apri il file corrispondente nella cartella `arguments/`
3. Ogni file contiene il testo completo della sezione con sotto-sezioni

**Note tecniche:**
- Il testo è stato estratto via OCR da un PDF scansionato: possono esserci piccole imperfezioni
- Le formule sono riportate in blocchi di codice per leggibilità
- La cartella `artifacts/ocr_raw/` contiene i file OCR grezzi (un file .txt per pagina del PDF)
